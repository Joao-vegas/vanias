# Brand System — Vânia Amaral

Sistema de marca da Vânia Amaral (Fisioterapia & Estética).

Esta pasta reúne os elementos de identidade visual:

- **Logo** — versões principal, monocromática e ícone
- **Paleta de cores** — blush / rosé + plum (tons usados no site)
- **Tipografia** — Cormorant Garamond (títulos) + Poppins (texto)
- **Assets** — fotos, ícones e elementos gráficos
- **Guidelines** — diretrizes de aplicação da marca

## Tipografia

- **Títulos:** Cormorant Garamond (serif, elegante)
- **Texto:** Poppins (sans-serif, leve)
