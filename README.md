# Vânia Amaral — Fisioterapia & Estética

Repositório do projeto da Vânia Amaral.

## Estrutura

```
.
├── site/           # Site institucional em Next.js 15 (App Router + Tailwind v4)
└── brand system/   # Sistema de marca (identidade visual, assets, guidelines)
```

## Site

Aplicação Next.js com landing page de captação (fisioterapia, estética e
atendimento domiciliar em São Paulo).

```bash
cd site
npm install
npm run dev      # http://localhost:3000
npm run build && npm run start
```

Conteúdo, contatos e textos centralizados em `site/lib/site.ts`.

## Brand System

Pasta destinada à identidade visual: paleta de cores, tipografia, logo,
diretrizes de uso e assets de marca.
